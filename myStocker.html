myStocker

Project Details

  • Category: Web Application
  • Project duration: Jan 2020 - Apr 2020

myStocker is a management platform for both brokers and traders. Traders can place their trades and brokers can approve and reject them accordingly with ease.

The platform enables:
  1. Quick and simple transactions via seamless buy and sell processes.
  2. Real-time visualizations of stock prices to aid easy reference.
  3. A user portfolio to provide an overview of all stocks purchased with one look.

The entire web application is lightweight and easily adaptable, as it relies on a series of microservices for the backend, and uses Vanilla JavaScript for the frontend with a template.

Tools & Technologies Used:
  1. CanvasJS
  2. Dockerized AMQP
  3. AlphaVantage API
  4. PayPal API
  5. News API
  6. Twilio
  7. Python Flask
  8. MySQL
