Capstone Project - Innovating the fruit trade industry

Project Details

  • Category: Data Analysis & Software Development
  • Company: Dimuto
  • Project duration: Dec 2020 to May 2021

DiMuto is a trade technology service championing Collaborative Commerce. They provide end-to-end supply chain visibility for global businesses, ensuring the traceability and trackability of business documents, goods and services with every transaction on their platform.

Dimuto is trying to revolutionise the fruit trade industry to remove the pain-points that make financing difficult.

Three main pain-points:

  1. Tedious insurance claim and purchase process.
  2. Inability to assess risk due to unpredictability of shipping fresh produce.
  3. Tedious and subjective quality assurance process that should protect both suppliers and buyers.

Our solution consists of a user friendly application incorporated with machine learning models to help :

  • Facilitate the process of settlement between buyer and seller.
  • Facilitate buyer/seller in purchasing insurance using quality scores of fruits.
  • Generate a quantitative quality score to measure the quality of avocados without human subjectivity.

These are some examples of the tools used:
Technology & Tools Description
Tensorflow & ImgAug Open source libraries used for training and finetuning EfficientDet classification and object detection models.
OpenCV, LabelImg, Object Detection Metric Software Python Packages for image preprocessing and model evaluation.
Django Python framework for backend.
AWS S3 Bucket, RDS, Elastic beanstalk AWS services used for file storage, database storage and deployment.

In conclusion, we managed to create:

  1. One object detection and one classification model with at least 86% accuracy wth IOU threshold of 80%.
  2. Machine learning models implemented into the backend for users to upload images which outputs quantity and quality of fruit, specifically avocado.
  3. Proof-of-concept web application with insurance application, approval and settlement application for supplier and buyers.
