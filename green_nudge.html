Social Analysis of Green Nudge's Social Media Presense

Project Details

  • Category: Data Analysis
  • Company: Green Nudge
  • Project duration: Aug 2020 to Nov 2020

Green Nudge is a green business that started in late 2017 which aims to raise awareness of environmental issues, encourage sustainable behaviour and drive collaboration among stakeholders. Green Nudge is mainly focused on providing services to companies to help facilitate sustainability in waste management generated by corporate events or within offices.

Green Nudge wants to improve engagement as they have not been seeing success in their efforts. As such, Green Nudge presented us with a problem statement: “How can Green Nudge reach out to more users on their social media platforms?”

We came up with 3 different Analysis: Hashtag Analysis, Content Generation, and Post Like Analysis. The objective was to increase engagement on their social media sites, expand their presense outside of Instagram and Facebook and lastly, identify if an echo chamber exists within their network respectively.

These are some examples of the tools used:
Technology & Tools Description
Chromedriver & Selenium Open source web automation tool that works with chrome browser that was used for data scrapping from Green Nudge's Instagram.
Tweepy API A Python library for accessing the Twitter API to scrape tweets relating to environmental topics and twitter users' sentiment.
Python Reddit API Wrapper/PRAW A Python package that allows access to Reddit's API for scrapping posts and subreddits discussing environmental topics.
Natural Language ToolKit/NLTK NLTK is a suite of python libraries and programs for symbolic and statistical natural language processing. NLTK was used for preprocessing of twitter and reddit data.
Topic Modelling with Latent Dirichlet Allocation LDA is a form of topic modelling and is used to classify text in a document to a particular topic. This helped us identify the main topics that reddits and twitter users were interested in.
Sentiment Analysis using VADAR Vader Lexicon is specifically attuned to sentiments expressed in social media. It was used to analyse tweets in greater detail based on the sentiment scores outputted.
Word Cloud We used python's word cloud generator to visualise the common hashtags used by Green Nudge as well as its competitors.
Tableau Tableau Software is an interactive data visualization software which we used to create a dashboard with easily visualised insights and can be effortlessly nagivated.

In conclusion, we managed to identify:

  1. Optimal number of hastags and the types of hashtags to use for maximal engagement
  2. Various topics and content that competitors have used as well as topics on reddit and twitter that will be useful for Green Nudge's content generation.
  3. The demographics of Green Nudge's community on Instagram as well as determining that there is no echo chamber present.
